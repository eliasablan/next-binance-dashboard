# Next Binance Dashboard – Real‑time Crypto Market Dashboard

## Español

### Next Binance Dashboard – Panel Web de Criptomonedas en Tiempo Real

Next Binance Dashboard es una aplicación web diseñada para ofrecer una experiencia centralizada y personalizada alrededor del mercado cripto. Permite visualizar precios y gráficos en tiempo real, explorar datos históricos por intervalos, analizar volumen y administrar una lista de criptomonedas favoritas, todo con una interfaz moderna y adaptable.

#### Características principales

- Precios y gráficos en tiempo real (velas y línea) a través de WebSocket de Binance.
- Históricos e intervalos configurables (1m–1M) y rangos de visualización (1h–1M).
- Gráfico de volumen con indicadores de cambio por vela.
- Buscador de criptomonedas con moneda base configurable (USDT, EUR, BTC, ETH).
- Lista de favoritas con acceso rápido y variación 24h en vivo.
- Tarjeta del activo seleccionado con precio actual.
- Preferencias persistentes: moneda base, tema (claro/oscuro) y favoritos.
- Interfaz receptiva y optimizada para móvil.
- Estado global para el panel, con vista expandida del gráfico y selección de tipo de gráfico.

#### Tecnologías utilizadas

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS 4 y shadcn/ui (Radix UI) para una interfaz moderna y accesible
- Recharts para visualizaciones interactivas
- Zustand para manejo de estado global
- next-themes para soporte de temas (claro/oscuro/sistema)
- nuqs para sincronización de parámetros en la URL (p. ej., símbolo)
- Librerías de apoyo: lucide-react, cmdk, vaul, sonner, usehooks-ts

## English

### Next Binance Dashboard – Real‑time Crypto Market Dashboard

Next Binance Dashboard is a web application built to deliver a centralized and personalized crypto market experience. It lets you visualize real‑time prices, candlestick and line charts, explore historical data by intervals, analyze volume, and manage a favorites list—all within a clean, responsive interface.

#### Key features

- Real‑time prices and charts (candlestick and line) via Binance WebSocket.
- Historical data with configurable intervals (1m–1M) and display ranges (1h–1M).
- Volume chart with per‑candle change indicators.
- Cryptocurrency finder with configurable base currency (USDT, EUR, BTC, ETH).
- Favorites list with quick access and live 24h change.
- Selected asset card showing the current price.
- Persistent preferences: base currency, theme (light/dark), and favorites.
- Responsive UI optimized for mobile.
- Global dashboard state with expanded chart view and chart type selection.

#### Technologies used

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS 4 and shadcn/ui (Radix UI) for a modern, accessible UI
- Recharts for interactive visualizations
- Zustand for global state management
- next-themes for light/dark/system theming
- nuqs for URL parameter sync (e.g., symbol)
- Supporting libraries: lucide-react, cmdk, vaul, sonner, usehooks-ts
